# Manual de usuario · Observatorio de Juventudes de Zipaquirá

Esta guía es para cualquier persona que **consulte** el tablero. No necesitas conocimientos técnicos.

## Abrir el observatorio

Entra al enlace publicado (GitHub Pages) o abre el archivo `index.html`. Verás el menú a la izquierda y los filtros arriba.

## El interruptor "Datos demo"

Arriba a la derecha. Mientras las entidades terminan de cargar las cifras reales, actívalo para ver el tablero con **valores de ejemplo** y entender cómo se verá cada gráfico. Desactívalo para ver el estado real (qué está y qué falta por diligenciar).

## Los filtros (afectan todo el tablero)

- **Año / Trimestre** — acota el período.
- **Secretaría** — una entidad específica.
- **Indicador** — un indicador puntual (la lista se ajusta a la secretaría elegida).
- **Género / Grupo etario / Zona** — actúan como un "lente": las cifras se recalculan para ese segmento.
- **Limpiar** — vuelve a ver todo.

## Las secciones del menú

| Sección | Qué encuentras |
|---|---|
| **Inicio** | Tarjetas KPI, análisis ejecutivo automático y el semáforo de diligenciamiento por entidad. |
| **Demográfico** | Composición por género, edad, zona y población diferencial. |
| **Secretarías** | Ranking de participación, nivel de reporte y evolución trimestral. |
| **Indicadores** | Tabla con buscador. Haz clic en los títulos de columna para ordenar. Exporta a CSV o JSON. |
| **Comparación** | Trimestre vs trimestre y entidad vs entidad. |
| **Alertas** | Avisos automáticos de variaciones fuertes, valores en cero, faltantes y brechas. |
| **Sala de Análisis** | Lectura inteligente: hallazgos, tendencias, riesgos, oportunidades y recomendaciones. |
| **Datos Abiertos** | Descarga de la base y consulta de fuentes y metadatos. |

## Exportar información

En **Indicadores** o **Datos Abiertos**, usa los botones **CSV** (para Excel) o **JSON**. La descarga respeta los filtros activos.

## El análisis ejecutivo

El texto en azul de "Inicio" y "Sala de Análisis" se **reescribe solo** cada vez que cambias un filtro, resumiendo en palabras lo que muestran los números.

## Consejo

Si una sección dice "Sin datos en la selección", probablemente el indicador aún no fue diligenciado para ese período. Revisa el semáforo de diligenciamiento en Inicio para ver qué entidades están al día.
